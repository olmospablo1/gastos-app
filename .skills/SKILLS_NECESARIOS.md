# Skills Técnicos para el Proyecto: Consultorio Gastos

### 1. Firebase Suite (Nivel Inicial)
* **Firestore:** Base de datos NoSQL para guardar gastos y cierres en tiempo real.
* **Firebase Auth:** Gestión de los 4 usuarios (Paola, Angie, etc.) para que nadie ajeno acceda.
* **Firebase Hosting:** Para que la App viva en una URL `.web.app` accesible desde el celular.

### 2. Desarrollo Frontend (Mobile First)
* **HTML5 & Tailwind CSS:** Estructura y diseño responsivo sin necesidad de escribir CSS complejo.
* **JavaScript ES6+:** Manejo de lógica, cálculos de saldos y manipulación del DOM.

### 3. Orquestación con Antigravity
* **Prompt Engineering:** Capacidad de traducir reglas de negocio a instrucciones técnicas.
* **Control de Versiones/Flujo:** Entender la secuencia lógica: Configurar -> CRUD -> Lógica de Cálculo -> Deploy.

### 4. QA & Testing
* **Validación de Reglas de Negocio:** Asegurar que (Gasto Total / 4) sea exacto.
* **Pruebas de Integración:** Verificar que el frontend hable correctamente con la API de Firebase.
